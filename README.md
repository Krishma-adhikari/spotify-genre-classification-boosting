# Spotify Genre Classification Using Boosting Algorithms

A machine learning project that compares **AdaBoost**, **Gradient Boosting**, and **XGBoost** for Spotify music genre classification.

## Dataset
Spotify Tracks Dataset (Kaggle)

## Models
- AdaBoost
- Gradient Boosting
- XGBoost

## Evaluation
- Accuracy
- Classification Report
- Confusion Matrix

## Results

| Model | Accuracy |
|--------|---------:|
| AdaBoost | 45.93% |
| Gradient Boosting | **70.67%** |
| XGBoost | 69.93% |

Gradient Boosting achieved the best performance on this dataset.
